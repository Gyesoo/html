[잡지/사보 이미지 폰트 분석 정리]

1. 첫 번째 텍스트 (웹은 지금 플래시 애니메이션)
- 사용 글꼴: 휴먼매직체 (또는 HY매직체)
- 특징: 매직펜으로 둥글게 쓴 듯한 손글씨 느낌. 오류 발생 시 굴림체로 대체될 수 있으니 정상적인 ttf 파일 설치 필수.

2. 두 번째 텍스트 (Special Issue)
- 영문: Special Issue
- 사용 글꼴: Comic Sans MS (Bold)
- 특징: 윈도우 기본 폰트. 캐주얼한 마커펜 느낌. 원본은 포토샵의 색상 오버레이(빨강), 경사 및 엠보스, 외부 광선(테두리) 효과가 적용됨.

3. 세 번째 이미지 (메인 타이틀)
- 영문 대형 로고: Web DESIgN
- 사용 글꼴: Futura (Bold 또는 Extra Bold)
- 특징: 'e', 'g'의 원형이 강조된 기하학적 폰트. 대/소문자를 섞어서 배치함.

- 한글 굵은 타이틀: 인프레스, 웹디자인
- 사용 글꼴: 윤고딕 100/300 시리즈 (또는 스포카 한 산스 네오 Bold, 맑은 고딕 Bold 로 대체 가능)

- 영문/한글 얇은 텍스트 (Internet / Web Design / 디지털 시대...)
- 사용 글꼴: Helvetica, Arial (영문), 윤고딕 얇은 웨이트 (한글)

4. 네 번째 이미지 (상단 웹주소 및 로고)
- 영문: www.impress.co.kr
- 사용 글꼴: Helvetica Bold 또는 Arial Bold

- 영문: Digital impress (로고 조합)
- 'Digital' 사용 글꼴: Brush Script MT (기본 붓글씨 폰트)
- 'impress' 사용 글꼴: Helvetica Black Oblique (또는 아주 굵은 고딕의 기울임꼴)

5. 다섯 번째 이미지 (년도 및 월)
- 영문 숫자(세로): 2001
- 사용 글꼴: Futura (Regular 또는 Medium) / 90도 회전하여 배치

- 영문 숫자(가로 굵게): 07
- 사용 글꼴: Helvetica Bold 또는 Arial Bold / 숫자 0 아래에 얇은 밑줄(Line) 추가

6. 여섯 번째 이미지 (책속의 책, digit 로고 등)
- 한글: 책속의 책
- 사용 글꼴: 윤고딕 100 시리즈 (중간 이상 굵기)

- 영문: Web Technic Book
- 사용 글꼴: Impact (좁고 아주 굵은 윈도우 기본 폰트)

- 영문 숫자 (바코드 아래): 7000
- 사용 글꼴: Arial Bold 등 기본 고딕

- 로고: digit
- 특징: 기성 폰트가 아닌 디자이너가 직접 그린 레터링(로고타입).

- 영문 얇은 텍스트: WEBSAFE T-SHIRTS \ 2001 \ AUTUMN...
- 사용 글꼴: Helvetica 또는 Arial (Light 또는 Thin 얇은 굵기)
- 특징: 자간(글자 간격)을 아주 넓게 띄워서 타이포그래피 연출.

- 색상 코드: 003333 등
- 사용 글꼴: 돋움(Dotum) 등 기본 고딕 폰트.
