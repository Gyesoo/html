🔤 주요 폰트 추정
================================================================

1. "Web DESIgN" 메인 타이틀 (가장 큰 영문)
----------------------------------------------------------------
- 추정 폰트: Bryant Bold 또는 Bryant Compressed Bold 계열
- 특징적인 단서:
  • 소문자 'g'가 single-story(한 층) 구조
  • 매우 굵은(Heavy/Black) 웨이트
  • 기하학적(geometric)이면서 살짝 응축된(condensed) 느낌
  • 'W'의 넓은 각도, 'S'의 둥근 형태
- 유사 후보: ITC Bauhaus, Bryant Pro, 
            Neutraface Condensed Display, Cachet


2. "07_2001" (오른쪽 상단 날짜)
----------------------------------------------------------------
- 메인 타이틀과 동일한 폰트 패밀리(Bryant 계열)로 보입니다.


3. "Internet / Web Design / Web Publishing..." (서브 텍스트)
----------------------------------------------------------------
- 추정 폰트: Helvetica 또는 Akzidenz-Grotesk 계열의 산세리프
- 클래식한 스위스 스타일 산세리프


4. "Special Issue", "WEBSAFE T-SHIRTS", "Web Technic Book"
----------------------------------------------------------------
- Helvetica / Univers 계열 산세리프로 추정


5. "digit" 로고
----------------------------------------------------------------
- 별도의 커스텀 로고타입(브랜드 전용 디자인)으로 보입니다.


6. 한글 "웹디자인", "임프레스", 
   "웹은 지금 플래시 애니메이션으로 술렁인다"
----------------------------------------------------------------
- 산돌 또는 윤디자인 계열의 고딕(Gothic) 서체
- 본문용은 윤고딕 또는 산돌고딕 계열로 추정
